## 1. 核心产品哲学：为什么要重新定义排名？
在台球（特别是中式台球、美式 9 球）领域，现有的排名系统（如 FargoRate, APA, 甚至传统的 ELO）在业余/半职业社交对局中存在三大痛点：
1. **反馈迟钝**：一场 7:6 的“险胜”和一场 7:0 的“零封”在传统系统中通常被同等对待，无法反映局内的实力压制。
2. **信任危机**：远程“刷分”、代打现象严重，缺乏物理层面的对局校验。
3. **社交缺失**：排名只是一个冰冷的数字，无法激发球馆内的竞争氛围和博弈乐趣。

## 2. BG-1 算法：数学模型与技术细节
NextUp-Rank 引入了 **Billiard-Glicko (BG-1)** 算法，基于 Glicko-2 进行深度特化：

### A. 局分胜率因子 ($S_{adj}$)
我们不再只看胜负。通过公式：
$$S_{adj} = 0.5 + \frac{W - L}{W + L} \times 0.5$$
- **7:0 完胜**：$S_{adj} = 1.0$ (最大积分收益)
- **7:6 险胜**：$S_{adj} = 0.538$ (接近平局，积分变动极小)
- **保护机制**：这有效防止了高分选手因偶尔状态起伏（惜败）而损失大量身价，也让完胜更有成就感。

### B. 三维球员定义 (Glicko-2 Standard)
1. **Rating ($r$)**: 核心实力估值。
2. **RD ($\phi$)**: 不确定度。新玩家 350；稳定玩家 < 50。**RD 会随不活跃天数自动衰减（升高）**。
3. **Volatility ($\sigma$)**: 波动率。使用 **Illinois Algorithm (标准 Glicko-2)** 迭代计算，精准捕捉选手的爆发式进步。

### C. 实时展示权重 ($\omega$)
引入记忆权重曲线：$\omega_i = e^{-\lambda(25 - i)}$。最近的 25 场比赛对当前分数展示影响最大，让用户在前端直观感受到“手感热、涨分快”。

## 3. CEO 战略：优化路径与防守逻辑
CEO 提出的五步走战略旨在构建一个“去中心化但高信任”的竞技场：

1. **LBS Handshake (地理握手)**：基于 PostGIS，双方直线距离需 < 200m 方可激活“现场认证赛”，否则降权 50% 或标记为友谊赛。
2. **Anti-Farming (防刷分)**：24 小时内同一对手超过 3 场比赛，积分收益指数级衰减至 0。
3. **Karma 系统**：建立“竞技信誉”。争议对局扣除 Karma，高 Karma 用户拥有裁决权。
4. **延迟奖励 (Global Recalibration)**：如果被你击败的对手随后表现神勇，系统会异步为你补发“奖励分”。
5. **Square Mode (广场模式)**：基于地理位置自动生成的球馆排行榜，赋予“球房霸主”、“狙击手”等社交称号。
